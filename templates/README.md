Templates

This folder contains the standardized document templates used throughout the Educator Intelligence Platform Design Book.

Maintaining consistent document structure improves readability, simplifies collaboration, and supports future automation for website generation, PDF publication, indexing, and citation management.

Available Templates

Template	Purpose
chapter-template.md	Standard format for book chapters
research-template.md	Literature review and paper summaries
adr-template.md	Architecture Decision Records
glossary-template.md	Terminology definitions

Philosophy

Every document in this repository should begin from a template rather than being created from scratch. Consistency allows the repository to evolve into a coherent scholarly work rather than a collection of unrelated notes.
